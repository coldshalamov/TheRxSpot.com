# TheRxSpot.com Project

## Overview
The Rx Spot is a telehealth and pharmacy platform website featuring:
- Patient services portal (weight loss, peptides, hair restoration, etc.)
- B2B healthcare provider access
- Business opportunities (pharma reps, affiliates, turnkey telehealth platform)
- Nutrition business partnerships

## Technology Stack
- **Frontend**: Static HTML/CSS/JS website
- **Forms**: FormSubmit.co for handling form submissions via email
- **Styling**: Custom CSS with CSS variables (design tokens)
- **Fonts**: Google Fonts (Inter, Outfit, Dancing Script)

## Key Files
- `index.html` - Main homepage with "Are You" feature boxes and modals
- `telehealth.html` - Telehealth platform onboarding form
- `nutrition-business.html` - Nutrition business interest form (NEW)
- `style.css` - Main stylesheet with design system
- `main.js` - Interactive components (modals, FAQ accordion, animations)
- `privacy-policy.html` - Privacy policy page
